Objective

Create a new subclass of Event called Personal by adapting the Program 9 starter code.

Context

You will start from the Event and WorkMeeting classes in Program 9 and create your own child class called Personal.

Your task is to:

  1. Keep the Event parent class
  2. Add a new child class called Personal
  3. Override one inherited method
  4. Add suitable private properties with getter and setter methods

Requirements

Inheritance Setup (FR1, FR2, FR9)

Create a child class called Personal that inherits from Event.

The constructor should:

  1. Accept the inherited event details
  2. Accept at least two new personal-event details
  3. Use super() to call the parent constructor
  4. Store the new child-class details as private attributes
class Personal(Event):
      def __init__(self, startDate, startTime, venue, reminder, occasionName, hostName):
        super().__init__(startDate, startTime, venue, reminder)
        self.__occasionName = str(occasionName)
        self.__hostName = str(hostName)

Overridden Method (FR3, FR7)

Override getDate() so the result is more suitable for a personal event.

def getDate(self):
      return "Personal event on " + super().getDate()

Getter and Setter Methods (FR4, FR5, FR6)

Create getter and setter methods for the new child-class properties.

def setOccasionName(self, newOccasionName):
      self.__occasionName = str(newOccasionName)

    def getOccasionName(self):
      return self.__occasionName

    def setHostName(self, newHostName):
      self.__hostName = str(newHostName)

    def getHostName(self):
      return self.__hostName
