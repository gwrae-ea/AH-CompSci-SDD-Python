Technical Explanation

This program builds on Program 7 by introducing overriding. Overriding happens when a child class creates a method with the same name as a method in the parent class. When that method is called on a child object, the child version runs instead of the parent version. In this example, WorkMeeting overrides getDate() so it can return a more specific result for a work meeting, while still inheriting the rest of the Event class.

Design

Parent class: Event Child class: WorkMeeting This program develops the inheritance example from Program 7 by adding an overridden method in the child class. The child keeps the inherited properties and methods from Event, but replaces the inherited getDate() method with a version that is more suitable for work meetings.

### English Pseudocode Parent Constructor: Event() 1. Accept the starting date, time, venue and reminder values. 2. Store each value inside the event object. 3. Create an empty participants list. 4. Set the participant index to the first free position. Parent Method: getDate() 1. Read the stored date from the object. 2. Return that date value. Child Constructor: WorkMeeting() 1. Accept all the event details plus the meeting title and mileage. 2. Call the parent constructor to set up the inherited event data. 3. Store the meeting title and mileage in the child object. 4. Set the initial travel expenses to zero. Child Method: getDate() 1. Call the parent version of getDate(). 2. Add extra text to show that the date belongs to a work meeting. 3. Return the new combined result. Child Method: calculateTravelExpenses() 1. Read the mileage stored in the child object. 2. Multiply the mileage by 1.5. 3. Store the result as the travel expenses. Child Method: getTravelExpenses() 1. Read the stored travel expenses. 2. Return that value.

Implementation

class Event:
    def __init__(self, startDate, startTime, venue, reminder):
        self.__startDate = str(startDate)  #String
        self.__startTime = str(startTime)  #String
        self.__venue = str(venue)          #String
        self.__reminder = reminder if isinstance(reminder, bool) else str(reminder).strip().lower() in ("true", "1", "yes", "y")    #Boolean (don't use bool(reminder): bool("False") is True)
        self.__participants = [""]*20 #Array of String
        self.__index = int(0)              #Integer

    def updateDate(self, eventDate):
        self.__startDate = str(eventDate)

    def getDate(self):
        return self.__startDate

    def addParticipant(self, name):
        self.__participants[self.__index] = str(name)
        self.__index = self.__index + 1

class WorkMeeting(Event):
  def __init__(self, startDate, startTime, venue, reminder, meetingTitle, mileage):
        super().__init__(startDate, startTime, venue, reminder)
    self.__meetingTitle = str(meetingTitle)      #String
    self.__mileage = float(mileage)              #Real
    self.__travelExpenses = float(0)             #Real

  def getDate(self):
    return "Work meeting on " + super().getDate()

    def calculateTravelExpenses(self):
    self.__travelExpenses = self.__mileage * 1.5
    
    def getTravelExpenses(self):
    return self.__travelExpenses
