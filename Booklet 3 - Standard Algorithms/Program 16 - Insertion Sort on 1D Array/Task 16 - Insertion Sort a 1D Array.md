# Task 16 - Insertion Sort a 1D Array

## Objective

Write an Insertion Sort function to sort a 1D array of numbers in ascending order.

---

## Context

You have an empty Python file: `Task 16 - Insertion Sort a 1D Array.py`.

Use Program 16 as the worked example and create your own version.

---

## Requirements

1. Create a function named `insertionSort(nA)`.
2. Start outer loop at index 1.
3. Store current value in `value`.
4. Shift larger previous values right while searching insertion point.
5. Insert `value` into the correct index.
6. Return and print sorted array.

---

## Suggested Tests

- Mixed values
- Nearly sorted values
- Reverse order values
- Duplicates
- Single value

---

## Functional Requirements Covered

- Standard Insertion Sort logic
- 1D array shifting and insertion
- In-place sorting
- Reusable sorting function
