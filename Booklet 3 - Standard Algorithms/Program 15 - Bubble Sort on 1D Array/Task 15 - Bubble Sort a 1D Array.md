# Task 15 - Bubble Sort a 1D Array

## Objective

Write a Bubble Sort function to sort a 1D array of numbers in ascending order.

---

## Context

You have an empty Python file: `Task 15 - Bubble Sort a 1D Array.py`.

Use Program 15 as the worked example and create your own version.

---

## Requirements

1. Create a function named `bubbleSort(nA)`.
2. Use adjacent comparisons (`nA[i]` and `nA[i+1]`).
3. Swap values when out of order.
4. Use a `swapped` flag to stop early if sorted.
5. Return the sorted array.
6. Print array before and after sorting.

---

## Suggested Tests

- Mixed values
- Already sorted values
- Reverse order values
- Duplicates
- Single value

---

## Functional Requirements Covered

- Standard Bubble Sort logic
- 1D array processing
- Controlled loop termination
- Reusable sorting function
