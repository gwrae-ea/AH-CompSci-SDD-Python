# Task 17 - Binary Search a Sorted Array

## Objective

Sort a 1D numeric array, then search for a target value using Binary Search.

---

## Context

You have an empty Python file: `Task 17 - Binary Search a Sorted Array.py`.

Use Program 17 as the worked example and create your own version.

---

## Requirements

1. Implement a sorting function (Bubble Sort or Insertion Sort).
2. Implement `binarySearch(nA, target)`.
3. Use `low`, `high`, and `mid` variables.
4. Return index if found, else return `-1`.
5. Ask user for target input.
6. Print sorted array and search result.

---

## Suggested Tests

- Target exists
- Target does not exist
- Target at first position
- Target at last position
- Duplicate target values

---

## Functional Requirements Covered

- Sort + search integration
- Binary Search control logic
- Index-based search result reporting
- Not-found handling
